# ImageReaderCV
The simple image reader with openCV and Qt.
Uses Qt 5.12 and openCV 4.6.0

The aplication  created to learn opencv tools.

The aplication supports the ability: 
 - output/save image;
 - change brightness; 
 - change  contrast;
 - gamma correction;

all features are implemented using opencv tools
